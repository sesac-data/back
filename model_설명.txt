match_agent instruction
 • 핵심 코드는 analyzer.py(지원금 매칭), file_parser.py(업로드 파일 파싱)



#01 Database 관련

config.py
    • 지원금 정책 DB 구조를 설정한 코드
        • 지원금 이름 정의
        • 지원금 정책 URL 정보
        • 지원금 정책 데이터(json, pdf) 이름 정의

db_builder.py
    • 지원금 정책 DB를 구축하는 코드
        • 방식: 
            - 지정된 경로에 pdf, json, docs, 등 파일이 없을 경우 지원금 url에서 크롤링하여 json으로 저장한다.
            - 여기서 파일이란 지원금 정책(지원 요건) 문서(pdf 등)이다.



#02 지원금 매칭 - 사용자 업로드 정보 파싱

file_parser.py
    • 사용자가 업로드한 인사 정보 파일을 파싱하는 코드
        • 방식:
            - GPT-4o를 통한 OCR



#03 지원금 매칭 - 지원금 분석

analyzer.py
    • 지원금 정책 DB와 사용자 업로드 정보를 비교하여 지원금 신청 가능 여부를 분석하는 코드
        • 방식: 
		    - GPT-5.4
			- 프롬프트를 통한 조건 상세 설정
			- 복잡한 지원금의 경우 전처리 함수를 통한 조건 검색 가이드 설정



#04 신청서 작성

mapping.py
    • 입력할 필드를 정의 및 PDF와 연결하는 코드

writer_final.py
    • 신청서 양식의 각 필드 별 좌표를 설정하고 입력한 값을 덮어씌워 PDF로 저장하는 코드
        • 라이브러리: 
            - ReportLab
            - PDFReader
